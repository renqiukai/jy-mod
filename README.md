# JYMod

一个基于 MFC 的小工具，用于读取/修改《金庸群侠传》存档（.GRP）中的主角坐标，并导出人物属性到 CSV。

## 功能

- 读取并修改存档中的主角坐标（X/Y）
- 预置地点快速跳转（下拉框 + 按钮）
- 打开 .GRP 存档后自动导出人物属性到 CSV
- 成员/人物信息查看与编辑对话框

## 环境与构建

- Windows + Visual Studio（MFC 应用）
- 打开 `JYMod.sln`，选择 Debug/Release 构建并运行

## 使用方式

1. 运行程序后点击“打开存档”，选择 `*.grp`
2. 在坐标输入框中修改 X/Y，或选择预置地点
3. 点击“保存位置”写回存档
4. 程序会在存档目录生成 `人物属性.csv`

## 注意事项

- 请先备份存档，程序会直接写入文件偏移
- 默认保存文件路径会记录在注册表中
- CSV 使用 UTF-8（含 BOM）以便 Excel 正确显示中文

## 目录说明

- `JYMod.sln` / `JYMod.vcxproj`：VS 工程
- `JYModDlg.*`：主界面逻辑（坐标读写/地点选择）
- `SaveGRPFile.*`：存档解析与 CSV 导出
- `res/`：资源文件
